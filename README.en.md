# Platformer_2d

**Platformer_2d** is a 2D platformer-style game project developed for the purpose of studying and practicing basic platform game mechanics, including movement, collisions, and interaction with the environment.

## Overview

The project implements classic 2D platformer mechanics, focusing on code organization, folder structure, and best practices in game development.

### Main Features

-   Basic character movement\
-   2D collision system\
-   Organized structure for project expansion\
-   Assets and scripts prepared for game evolution\

## Technologies and Tools Used

-   Game engine: Unity\
-   Language: C#\
-   Version control: Git

## How to Download and Run

1.  Clone this repository:

    ``` bash
    git clone https://github.com/rafacf95/Platformer_2d.git
    ```

2.  Navigate to the project directory:

    ``` bash
    cd Platformer_2d
    ```

3.  Open the project in Unity Hub using the appropriate Unity version.

4.  Press **Play** to run the game.
