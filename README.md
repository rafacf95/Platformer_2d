# Platformer_2d

**Platformer_2d** é um projeto de jogo 2D no estilo platformer,
desenvolvido para fins de estudo e prática de mecânicas básicas de jogos
de plataforma, incluindo movimentação, colisões e interação com o
ambiente.

## Visão Geral

O projeto implementa mecânicas clássicas de um platformer 2D, com foco
em organização de código, estruturação de pastas e boas práticas de
desenvolvimento de jogos.

### Características Principais

-   Movimento básico do personagem\
-   Sistema de colisão 2D\
-   Estrutura organizada para expansão do projeto\
-   Assets e scripts preparados para evolução do jogo

## Tecnologias e Ferramentas Utilizadas

-   Motor de jogo: Unity\
-   Linguagem: C#\
-   Git para controle de versão\

## Como Baixar e Executar

1.  Clone este repositório:

    ``` bash
    git clone https://github.com/rafacf95/Platformer_2d.git
    ```

2.  Acesse o diretório do projeto:

    ``` bash
    cd Platformer_2d
    ```

3.  Troque para a branch de desenvolvimento:

    ``` bash
    git checkout develop
    ```

4.  Abra o projeto no Unity Hub utilizando a versão correspondente à do
    repositório.

5.  Pressione **Play** para rodar o jogo.
